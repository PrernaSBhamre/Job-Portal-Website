# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision
To establish a functional Job Portal Website with a robust Node.js/Express backend, a public-facing vanilla HTML frontend, and a Vite-based administrative portal, enabling job seekers to find listings, employers to post positions, and admins to moderate platform content.

## Goals
1. Finalize the backend API functionality and endpoint robustness.
2. Complete integration between the frontend/admin-portal and the backend API.
3. Facilitate future updates, enhancements, and architectural refinement for the Job Portal.
4. Keep track of developments securely through centralized GSD principles.

## Non-Goals (Out of Scope)
- Complete rewrite of the existing technology stack.

## Users
- **Job Seekers**: Search, view, and apply for jobs.
- **Recruiters/Employers**: Create, manage, and track job postings.
- **Administrators**: Moderate content and manage the entire platform.

## Constraints
- **Technical constraints**: Must seamlessly coordinate the vanilla HTML frontend, Vite admin panel, and Express API.

## Success Criteria
- [ ] Mapped architecture fully captures the existing structure.
- [ ] GSD documentation serves as a single source of truth.
- [ ] Future update tasks are clearly tracked in ROADMAP and TODO.
