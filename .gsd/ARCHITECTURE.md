## Data Flow

1. Client (Frontend/Admin Portal) makes an HTTP request to the Backend API.
2. Backend API processes the request, validating authentication via JWT.
3. Controller interacts with Mongoose models to query or mutate data in MongoDB.
4. Response is sent back to the client in JSON format.
